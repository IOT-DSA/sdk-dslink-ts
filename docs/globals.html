useDsaChildQuery

  • useDsaChildQuery(node: NodeQueryResult, callback?: Listener<NodeQueryResult>, watchChildren?: "*" | string[]): void
    • Defined in react-hook.ts:91

    Query a child node and its children

    Parameters

    • node: NodeQueryResult

      The node from a result of a parent query.

    • Optional callback: Listener<NodeQueryResult>

      The callback will be called only when

      • node value changed if ?value is defined
      • value of config that matches ?configs is changed
      • value of attribute that matches ?attributes is changed
      • child is removed or new child is added when wildcard children match * is defined
      • an child has updated internally (same as the above condition), and the child is defined in watchChildren
    • Optional watchChildren: "*" | string[]

      defines the children nodes that will trigger the callback on any change

    Returns void
